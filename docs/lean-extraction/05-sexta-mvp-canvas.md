# Sexta-feira — MVP Canvas

## Visão do MVP

Para peritos em geral/compliance cujo interesse é identificar quem, o quê, quando e porquê determinada manipulação de dados ocorreu, o RavenLedger é um sistema de auditoria de dados/TI, via browser, que permite a identificação e localização de dados alterados por processos ou pessoas. Diferente do EntityFramework, nosso produto oferece uma interface amigável para o usuário e seus processos internos garantem a legitimidade dos dados auditados.
